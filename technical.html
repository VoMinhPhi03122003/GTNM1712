Firebase User Guide

What is Firebase? How to "deploy" a website to Firebase. Today group 6 will introduce Firebase and guide you how to deploy "HTML-CSS-JS" into firebase.

1. Introducing Firebase

Concept:

Firebase is a real-time database service provided by Google and operated in the cloud to assist in the creation of high-quality mobile and web applications. With using Firebase, Developers can focus on developing their apps without worrying about how their product will function. and is administered on the backend.

Advantages:
  • Deploying the application quickly: Firebase saves a lot of time managing and syncing all data for users. That's because the user is not under pressure, interested in the part supported, with good APIs and cross-platform support.
  • Safety: You can be completely assured of Firebase security thanks to the cloud platform, connecting over SSL, using JavaScript for database user rights, ...
  • Stable: Written on the cloud platform provided by Google, the tools always ensure maximum stability. Besides, the process of upgrading or maintaining the Server is also quicker and simpler.
    • Disadvantages:

    Firebase databases are organized in a tree, parent-child fashion. Meanwhile, SQL users are familiar with the traditional Table style. When using Firebase, it may take a while for users to get used to it before mastering it.

    DEPLOY FIREBASE



    1. Install Node js and Firebase CLI:


    First, check if your computer has NodeJS installed. Open command (Window+R >> cmd >> OK) and enter: node --version


    If cmd does not show the version of Node JS, proceed to install Node for the computer here.

    After installing Node Js, we proceed to install CLI for Firebase by typing the command " npm install -g firebase-tools " and cmd


    After installing, type the command: firebase --version, if the screen shows the version, the installation is successful.


    2. Steps to Deploy 1 Project to Firebase:


    B1: Need to create a folder on your computer to contain the project



    B2 : Open cmd in the folder just created .



    B3 :Sign in to firebase with the command " firebase login ".



    B4 : Continue typing the command " firebase init " to connect the CLI to your Firebase online account.



    B5 : Select " Use an existing project " and choose the name of the project you created above.


    B6 : Answer the questions like the image below and wait for "Firebase initialization complete!" to appear.



    B7 : Next, you need to type the command " firebase deploy " to put the code on firebase's hosting.


    B8 : Visit the link in the "Project console" or "Hosting URL" section to access your website.
